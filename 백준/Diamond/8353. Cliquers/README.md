### 문제 설명

<p>An undirected graph is called <b>a cliquer</b> if each connected component of the graph is a clique. Maurycy has drawn all cliquers with <em>n</em> vertices on a piece of paper and is going to assess beauty of each of them with a number from the set {1, ..., <em>m</em>} (in particular, different cliquers may be assigned equal grades). In how many ways can he do this? The result should be computed modulo 10<sup>9</sup> - 401. The figure below depicts all cliquers for <em>n</em> = 3.</p>

<p style="text-align: center;"><img alt="" src="https://upload.acmicpc.net/4a1211f5-c2bb-4629-9d70-540ab275d270/-/preview/"></p>

### 입력 

 <p>The only line of the standard input contains two integers <em>n</em> and <em>m</em> (1 ≤ <em>n</em>, <em>m</em> ≤ 200 000), separated by a single space and denoting the number of vertices of each cliquer and the number of grades respectively.</p>

### 출력 

 <p>The only line of the standard output should contain the number of possible sets of grades modulo 10<sup>9</sup> - 401.</p>
