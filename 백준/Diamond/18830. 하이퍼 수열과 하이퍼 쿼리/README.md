<p>둘째 줄부터는 <em>A</em><sub>11111111111</sub>, <em>A</em><sub>11111111112</sub>, ..., <em>A</em><sub><em>m</em><em>n</em><em>o</em><em>p</em><em>q</em><em>r</em><em>s</em><em>t</em><em>u</em><em>v</em></sub><em><sub>w</sub></em>가 아래와 같이 주어진다.
